• About Apna Ghar

    Apna Ghar means Own-Home in English we are a platform which aim to make lives of the Old and needy easier, we are a system which would allow users to connect with old age homes relative to the geographical location of the user, we have an SoS system which can be used in case of any emergency. Our platform also has a donation system in place which can allow users to donate and change lifes of the old and needy. We are there to protect the innocent and to shelter the weak and needy.

    FAQ

    Do you only help the old and needy?

    Yes, we aim to only help the old and needy.

    What is SOS?

    SOS or Save Our Soul is a system which allows us to react to any emergencies it is basically an emergency button which sends us the location where help is needed.
